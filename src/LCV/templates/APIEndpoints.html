Example of usage of the API methods: /LicensesInput

  • http://127.0.0.1:3251/LicensesInput?InboundLicenses=Apache 2.0,MIT License&OutboundLicense=MIT

This endpoint accept a POST request with the following syntax:

  • /LicensesInput?InboundLicenses=FirstInboundLicense,SecondInboundLicense; ... nInboundLicense&OutboundLicense=UniqueOutboundLicense

Please use Postman to first verify that you are using the right syntax:

This is an example of POST:

  • http://lima.ewi.tudelft.nl/lcv/LicensesInput?InboundLicenses=The Apache Software License, Version verbose license name 2.0;not within aliases GPL 3.0;LGPL not alias 2.1&OutboundLicense=GPL never been an alias 3.0

Please note that the separator used between inbound licenses is column `;`, and not anymore comma `,` (because many aliases have comma in the middle).
